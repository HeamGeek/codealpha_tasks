# Iris Flower Classification – CodeAlpha Internship

## Objective
To classify Iris flowers into Setosa, Versicolor and Virginica using machine learning.

## Dataset
Iris flower dataset containing:
- Sepal length
- Sepal width
- Petal length
- Petal width

## Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn

## Model Used
- Random Forest Classifier

## Results
- Accuracy: ~95–100%
- Strong classification performance

## How to Run
```bash
python iris_classification.py
