# Car Price Prediction – CodeAlpha Internship

## Objective
Predict the price of a used car based on attributes such as year, mileage, fuel type, transmission and engine specifications.

## Dataset
Car price prediction dataset downloaded from Kaggle. 

## Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## Model
Random Forest Regressor with preprocessing pipeline.

## Results
- MAE, RMSE, and R² score reported.
- Scatter plot comparing actual vs predicted prices.

## How to Run
```bash
python car_price_prediction.py
